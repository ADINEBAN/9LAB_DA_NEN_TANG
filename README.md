# 9 Flutter Labs

This workspace contains 9 separate Flutter lab folders based on the assignment in `Báo cáo 9 Lab_ DANENTANG.docx`.

Folders:

- `lab_01_iam_rich`
- `lab_02_micard`
- `lab_03_dice`
- `lab_04_magic_8_ball`
- `lab_05_xylophone`
- `lab_06_quizzler`
- `lab_07_destini`
- `lab_08_bmi_calculator`
- `lab_09_clima`

Notes:

- The machine used to create this submission does not have `flutter` or `dart` installed, so the apps were authored manually instead of scaffolded by `flutter create`.
- Each folder contains a minimal Flutter app with `pubspec.yaml` and `lib/main.dart`.
- `lab_05_xylophone` focuses on the UI and interaction flow. It uses `SystemSound.click` instead of external audio assets.
- `lab_09_clima` is implemented as a polished weather UI with mocked city data so it stays runnable without an API key or extra packages.

When Flutter is installed, run inside any lab folder:

```bash
flutter pub get
flutter run
```
